# Getting Started

Install the CLI, create a project, and publish your first site.

## Install the CLI

Download the release for your platform and verify that `boris --help` runs.

## Build a first site

Run `boris` from a project containing `content/` and a layout. The default
output is a self-contained static tree under `dist/`.
