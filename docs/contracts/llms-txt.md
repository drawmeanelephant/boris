# `llms.txt` export contract

**Status:** v0.6 first slice
**Format:** `llms.txt`

Boris can emit a deterministic, human-readable `llms.txt` projection of a
validated content graph. It is an export for crawlers and language-model
clients; it is not a replacement for HTML, IR, RAG, or the context bundle.

## CLI

`boris --llms` writes `llms.txt` at the repository root. Use
`boris --llms-path PATH` to choose another relative path. The path is treated
as an output file, not as a content or URL root; absolute paths are rejected.
The mode cannot be combined with HTML, IR, RAG, or context selectors.

The exporter calls the shared `pipeline.compile` path. Invalid content leaves
the previous output untouched and returns the normal content failure code.

## Format

The first slice emits:

```markdown
# Boris documentation

> This file is generated by Boris from a validated Trunk/Satellite content graph.

## Documentation

- [A page](/a-page/): First useful paragraph from the page.
  - [A child](/a-child/): First useful paragraph from the child.
```

Rules:

- The heading and provenance line are fixed and contain no timestamps or host
  data.
- Pages are emitted in deterministic graph order: roots by canonical entity
  id, then each direct child recursively by canonical entity id.
- Every page has a Markdown link whose title is frontmatter `title`, falling
  back to its canonical entity id.
- The URL fallback is `/<entity-id>/`. Boris does not currently accept an
  author URL field, so deployment-specific URL rewriting belongs outside this
  first slice.
- The description is the first non-heading paragraph after frontmatter,
  whitespace-normalized and bounded to 240 bytes. When no paragraph exists,
  the page title is used.
- Link punctuation and control whitespace are escaped or normalized so page
  metadata cannot break the list structure.
- The output is published through a sibling staging file. A failed write or
  rename does not replace the previous complete export.

`llms-full.txt` and deployment-aware URL configuration are deliberately later
work; they must not silently change this first-slice contract.
