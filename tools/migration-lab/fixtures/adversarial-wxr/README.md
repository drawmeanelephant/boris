# adversarial-wxr fixture

Compact hostile WXR corpus: Unicode, same-type duplicate slugs, nested pages,
encoded unresolved links, raw HTML/shortcodes/gallery, duplicate media
basenames, and an unsupported custom post type. The two local `hero.png` files
exercise basename ambiguity; the gallery remains unavailable to offline input.

For WPTT Theme Unit Test–class gaps (statuses, comments/trackbacks, post
formats, menus, empty/long titles, taxonomy cardinality), see
[`../wptt-derived/`](../wptt-derived/).
