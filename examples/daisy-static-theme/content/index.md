# A friendly static docs surface

This small site demonstrates a polished reading layout using only Boris
Markdown, a local theme, and its existing closed layout slots.

## Start here

| Page | What it demonstrates |
|---|---|
| [Foundations](foundations.md) | A Trunk with a generated direct-child list |
| [Patterns](patterns.md) | Code, tables, and an Aside in a Satellite page |

<Aside kind="tip">

The soft cards and badges are hand-authored CSS. They are a visual pattern,
not a framework dependency or a new Boris default.

</Aside>

## Reading rhythm

The wide screen layout separates site navigation, article content, and the
page outline. On a narrow screen it becomes one comfortable reading column.
