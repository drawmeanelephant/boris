# Foundations

Use a small token set and obvious hierarchy before adding decoration. This
Trunk page intentionally has a child, so `{{children}}` renders a direct link
to the contrast note.

## Practical defaults

- Keep body text on a quiet surface.
- Reserve the strongest color for links and focus.
- Let layout columns collapse before text gets cramped.

<Aside kind="note">

The navigation, breadcrumb, TOC, and direct-child list are emitted by Boris
from the validated graph. This theme only supplies their surrounding HTML and
CSS.

</Aside>
