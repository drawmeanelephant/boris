# Small rules for useful docs

This independent theme prototype explores a compact documentation surface for
Boris: quiet borders, readable defaults, and enough structure to find the next
useful page without a framework runtime.

## A small visual system

| Choice | Prototype treatment |
|---|---|
| Layout | fluid columns that collapse to one reading column |
| Type | system sans with compact scale and generous line height |
| Color | paper-like surfaces with a green action color |
| Components | native HTML, local CSS, and Boris-generated slots |

<Aside kind="tip">

The inspiration is Chota's compact, utility-light spirit. The CSS, content,
layout markup, names, and visual decisions in this folder are original to this
example.

</Aside>
