# Accessibility notes

- The layout uses `header`, `nav`, `main`, `article`, `section`, `aside`, and
  `footer` landmarks.
- The userpic is text with an explicit accessible label, not a decorative image.
- Links retain visible underlines and receive a high-contrast keyboard focus
  ring.
- The green-on-black palette is intentionally high contrast; verify it again if
  the theme is recolored.
- Small screens collapse the entry header without hiding content or navigation.
- Reduced-motion preferences disable transitions.
